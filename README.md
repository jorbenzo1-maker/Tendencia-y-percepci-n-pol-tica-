# Tendencia y Percepción Política

Repositorio orientado al análisis de tendencias políticas, percepción pública, comunicación estratégica e inteligencia artificial aplicada al entorno electoral.

## Objetivos
- Analizar comportamiento político digital
- Estudiar percepción pública
- Explorar automatización y análisis de datos
- Integrar inteligencia artificial al marketing político

## Futuras líneas de desarrollo
- Dashboards electorales
- Scripts en Python
- Análisis de sentimiento
- Monitoreo de redes sociales
- Modelos predictivos
